    - iGPU:         Intel HD 4600 1536MM
    - Mainboard:    HM86
    - Bluetooth:    Atheros AR3012 Bluetooth 4.0
    - Wifi:         Atheros AR9462 Wireless
    - eGPU:         Nvidia GT845M
    - USB:          3.0 Standard Type A
    - Audio:        Realtek ALC668
